## About The Project
 
  
     Create a function that repeats each character in a string n times.
     
     Examples:
     repeating("mice", 5) ➞ "mmmmmiiiiiccccceeeee"
     repeating("hello", 3) ➞ "hhheeellllllooo"
     repeating("stop", 1) ➞ "stop"
